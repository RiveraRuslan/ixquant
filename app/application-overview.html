Application overview

Our Option analysis set of tools is based on the Black-Scholes option pricing model. The application is built in such a way that it is not linked to the current real market data but rather to the theoritical option values calculated by the option pricing model. This gives you an unlimited flexibility in building and simulating the option trading strategies and quickly see how they would perform under the various market metrics.

Components of the Option Analysis application:

Option Simulator

The Option Simulator screen will be your main working environement for option analysis and strategies testing. Multiple features and controls are available. To name just some of them:

  • Set up the desired market metrics for your simulation
  • Build multi-leg and multi-expiration strategies
  • Express the simulation results in the currency of your choice
  • Quickly visualise the simulation results with the Payoff chart

Setups DB

When making a strategy simulation, you will be able to save it for future use. The Setups DB is the database of all the setups (strategies) or portfolios you created and saved in the Option Simulator.

  • With one click a setup or a portfolio is uploaded into the Option Simulator window and you can work with it again
  • Sort the saved setups by the underlying instrument or the strategy bias (bullish, bearish or neutral)
  • Delete the unnedeed setups or portfolios

Instruments table

The Instruments table is the database of all your underlying instruments you are working with. You will be able to manage and adjust this list according to your needs:

  • Define which underlying instruments you want to work with
  • Set up the market metrics for each underlying instrument (i.e. IV, expiry dates, options related data and so on)
  • It is also possible to restore the data to default levels with one click
